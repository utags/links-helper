## Usages

### Examples of rules that open links in a new tab

#### Youtube

Video player page

```js
^/watch
^/shorts
```

#### V2EX

Subject page, member page, settings page

```js
^/t/\d+(#.*)?$
^/member/[^?]*$
^/settings/
```
